    📈 상권 자동 판단 🤖 자동

    위치·반경·업종을 기준으로 자동으로 조회됩니다. 직접 수정할 수 없습니다.

    💰 임대 조건 ✏️ 직접 입력

    실제 계약 금액 또는 예정 금액을 입력하세요. 판단의 핵심 지표입니다.

    초기 자본 회수 기간 계산에 사용됩니다.

    👥 인력 및 인건비 ✏️ 직접 입력

    주 15시간 이상 근무 예정 파트타임 인원을 입력하세요.

    점주 본인도 포함해 입력하세요.

    직접 입력 시 인원 수 기반 추정보다 정확한 판단이 가능합니다.

    ⚙️ 고급 설정 🔧 선택 입력하면 판단 정확도가 높아집니다

    🏢 상가 상세 정보

    1층 외 층수는 유동인구 판단에 영향을 줍니다.

    전용 주차 공간이 있으면 체크하세요.

    ⚙️ 매출·수익 지표
